# Vue TypeScript Boilerplate
Custom Linting configuration for Vue 3 with TypeScript Structure

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
